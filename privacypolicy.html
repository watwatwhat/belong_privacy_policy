プライバシーポリシー

BELONG を安心してご利用いただくために

BELONG Team（以下，「当社」といいます。）は，アプリ「BELONG」（所有品を一元管理するためのモバイルアプリ。以下，「本サービス」といいます。）における個人情報およびユーザーデータの取扱いについて，以下のとおりプライバシーポリシー（以下，「本ポリシー」といいます。）を定めます。

第1条（取得するデータの種類）

本サービスが取得する主なデータは以下のとおりです。

  • アカウント情報：メールアドレス，OAuth プロバイダから受領する識別子や表示名等（Supabase が管理）
  • 所有品データ：タイトル，カテゴリ，保管場所，数量，価格，メモ，ステータス（owned / wanted / other），タグなどユーザーが入力する情報
  • カテゴリ・場所情報：名称，アイコン，階層情報，カラー等（初期データまたはユーザー入力）
  • バックアップデータ：ユーザーがエクスポートする JSON 文字列（ユーザー自身で保存・管理）
  • 広告・分析用データ：将来的に AdMob など外部 SDK が取得する識別子や操作ログ（機能が有効化された場合）

現時点で，画像・正確な位置情報・連絡先などの機微データは取得していません。

第2条（取得方法と利用目的）

当社は，ユーザーがアプリに入力・操作することでデータを取得します。取得したデータは次の目的で利用します。

  • 所有品の記録，検索，カテゴリ・場所管理，統計表示など本サービスの主要機能を提供するため
  • バックアップやデータ移行などユーザー主導のデータ管理機能を提供するため
  • 認証・不正利用防止・問い合わせ対応など，安全で安定したサービス運営を行うため
  • 将来提供予定のクラウド同期や広告機能において，ユーザーの同意に基づき最適化を行うため

第3条（保存先と安全管理措置）

本サービスは，所有品データ・カテゴリ・場所情報を端末内の AsyncStorage に暗号化せず保存します。端末の紛失・盗難時に第三者へ閲覧される恐れがあるため，OS の画面ロックやデバイス暗号化の利用を推奨します。

アカウント情報は Supabase のクラウド環境で管理し，TLS により通信を暗号化します。アクセストークンやリフレッシュトークンは同じく AsyncStorage に保管されます。

第4条（第三者提供）

当社は，ユーザーの同意または法令に基づく場合を除き，個人情報を第三者に提供しません。なお，認証・広告など外部サービスを利用する機能においては，各サービスに必要な最小限の情報が送信されます。

将来的にクラウド同期や広告 SDK を有効化する場合，取得・送信される情報の範囲と目的をアプリ内で明示し，必要に応じて同意管理を行います。

第5条（ユーザーの権利）

ユーザーは，次の権利を有します。

  • アプリ内で所有品データやカテゴリ情報を閲覧・編集・削除する権利
  • Supabase アカウントに紐づく個人情報の開示・訂正・利用停止・削除を当社へ請求する権利
  • バックアップ機能を用いたデータのエクスポート／インポートを行う権利

お問い合わせを受領した場合，合理的範囲で速やかに対応します。ただし，法令上保存義務のある情報や技術的に削除が困難な情報については，対応を制限する場合があります。

第6条（データの保存期間と削除）

端末内に保存されたデータは，ユーザーが削除操作を行うまで保持されます。アプリをアンインストールしてもデバイスのバックアップサービスによって復元される場合があります。

Supabase が保持するアカウント情報は，ユーザーが退会または削除申請を行った際に順次削除します。バックアップファイルはユーザー自身の責任で管理してください。

第7条（広告・解析ツールの利用）

本サービスは AdMob を将来的に導入する設計ですが，現時点では SDK を有効化していません。広告を配信する際は，Google モバイル広告ポリシーや各国の法令に基づきコンセント取得・年齢層設定等を行います。

第8条（ポリシーの改定）

当社は，法令の改正や機能追加などに応じて本ポリシーを改定することがあります。重要な変更がある場合は，アプリ内通知または Web サイト等で周知し，必要に応じて再同意を取得します。
